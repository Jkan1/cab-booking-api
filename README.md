# Cab Booking API

* A collection of APIs for cab booking system with Swagger Documentation.
* **Modules**
    * Customer
    * Driver
    * Admin

* **Booking Status**
    * 0 : PENDING,
    * 1 : DRIVER_ASSIGNED, 
    * 2 : COMPLETED, 
    * 3 :	CANCELLED 
 											
* **Driver Status**
    * 0 : FREE,
    * 1 : ASSIGNED, 
    * 2 : NOT_AVAILABLE
